## Color

Company brand card applied to a **60 / 30 / 10** light-glass system. Researcher decision: use the **company palette** (not the old washed teal) — deep blue reads as "trustworthy / professional," correct for a patent-legal tool, while the teal preserves brand continuity.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#F3F5F3` + tinted gradient mesh | App canvas behind all glass |
| Secondary (30%) | frosted white glass `rgba(255,255,255,0.60)` + blur | Sidebar, topbar, cards, panels |
| Accent (10%) | `#0B3D91` (deep blue) | See reserved list below |
| Brand / support | `#016C51` (teal) | Logo mark, links, selected/active item tint |
| Success (semantic) | `#1DBF73` | Passed gates, completed steps — **fill/indicator only** |
| Destructive | `#D92D20` (palette has no red — researcher-added) | Delete / irreversible actions only |

**Accent `#0B3D91` reserved for (explicit — never "all interactive"):**
1. Primary CTA buttons (solid blue, white text)
2. Active nav item indicator + active tab underline
3. Focus rings (`:focus-visible`)
4. Wizard progress-bar fill + current-step marker
5. Selected-state ring on choice cards

**Canvas mesh (the backdrop the frost refracts):**
```css
background-color: #F3F5F3;
background-image:
  radial-gradient(at 0% 0%,   color-mix(in oklch, #0B3D91 16%, transparent) 0, transparent 55%),
  radial-gradient(at 100% 0%, color-mix(in oklch, #016C51 14%, transparent) 0, transparent 55%),
  radial-gradient(at 50% 100%,color-mix(in oklch, #1DBF73 8%,  transparent) 0, transparent 60%);
```
Keep tints soft/desaturated so text contrast holds over glass.

**Text tokens (on light glass):**

| Token | Value | Contrast on `rgba(255,255,255,0.60)` glass |
|-------|-------|--------------------------------------------|
| `--text-primary` | `#0B1B2B` | ~14:1 ✓ |
| `--text-muted` | `#46586B` | ~6.5:1 ✓ |
| `--text-soft` | `#6B7A89` | ~4.6:1 ✓ |

**Contrast guardrails (verified):**
- `#0B3D91` solid button + white text → ~10:1 ✓
- `#016C51` as text/icon on white glass → ~5.8:1 ✓
- ⚠️ `#1DBF73` as text on white glass → **~2.2:1 FAIL.** Never use bright green for text/icons on light. Use it only as a **fill** (badge bg with dark text) or a **status dot / progress fill**. For success *text*, use `#016C51` or darkened `#15803D`.

## Glassmorphism Tokens

Light-tuned overrides of the standard glass kit (standard kit is dark-tuned; on a light canvas, glass must sit at higher white opacity so text stays legible).

| Token | Value |
|-------|-------|
| `--glass-bg-light` | `rgba(255,255,255,0.45)` |
| `--glass-bg` | `rgba(255,255,255,0.60)` |
| `--glass-bg-heavy` | `rgba(255,255,255,0.72)` |
| `--glass-blur-light` | `blur(8px)` |
| `--glass-blur` | `blur(16px)` |
| `--glass-blur-strong` | `blur(28px)` |
| `--glass-border` | `1px solid rgba(255,255,255,0.55)` (top-edge highlight) |
| `--glass-hairline` | `inset 0 0 0 1px rgba(15,40,60,0.06)` (definition on light) |
| `--glass-shadow` | `0 4px 24px rgba(11,61,145,0.08)` (brand-tinted) |
| `--glass-shadow-elevated` | `0 12px 40px rgba(11,61,145,0.12)` |

**Surface roles:**
- **Sidebar / topbar:** `--glass-bg-heavy` + `--glass-blur-strong`, sticky.
- **Panels / cards:** `--glass-bg` + `--glass-blur` + `--glass-shadow` + `--glass-hairline`. **Replace the old "1px border + subtle fill on everything"** — glass elevation (blur + soft shadow) is the new separator. Reserve hard borders for inputs and thin dividers only.
- **Ghost buttons / chips:** `--glass-bg-light` + `--glass-blur-light`.

**Mandatory fallbacks (from glassmorphism a11y spec):**
```css
@supports not (backdrop-filter: blur(1px)) {
  .glass { background: rgba(255,255,255,0.92); }
}
@media (prefers-reduced-transparency: reduce) {
  .glass { background: rgba(255,255,255,0.94); backdrop-filter: none; }
}
@media (prefers-contrast: more) {
  .glass { background: rgba(255,255,255,0.90); border-color: rgba(15,40,60,0.30); }
}
```
Always pair `backdrop-filter` with `-webkit-backdrop-filter` (Tauri ships Chromium, but keep it portable).

## Layout & Interaction Contract (rework)

**App shell** — three zones over the mesh canvas:
1. **Glass sidebar** (`260px`, sticky, `--glass-bg-heavy`): brand mark in `#016C51`; grouped nav (开始 / 项目 / 设置 + expert tool groups 知识库·发明点·交底与策略·质检·导出). Active item: blue indicator bar + teal-tinted glass fill. Collapses to a shadcn `Sheet` under 920px.
2. **Glass topbar** (sticky, `--glass-bg-heavy` + strong blur): page title (Display) + subtitle + contextual action cluster (right-aligned) + project picker + theme toggle.
3. **Workspace**: glass panels on the mesh; max content width `1120px`; `lg`/`xl` rhythm.

**Guided wizard (primary flow) rework:**
- Horizontal **glass stepper rail**; current step = blue marker, done = `#1DBF73` check fill, locked = reduced opacity.
- Progress bar fill in `#0B3D91`.
- Each step = one glass panel; primary action docked bottom-right.
- Start screen: 3 glass choice cards (发明 / 实用新型 / 导入润色) with lucide icon in teal, hover lift `translateY(-2px)`.

**Motion:** transitions 120–200ms ease; hover lift on cards/buttons; `@media (prefers-reduced-motion: reduce)` disables transforms.

**Density:** keep information-dense (it's a pro tool) but breathe — replace border-boxing with glass grouping + whitespace.

## Copywriting Contract

App is Chinese (`zh-CN`). Copy stays Chinese.

| Element | Copy |
|---------|------|
| Primary CTA | **开始撰写** (verb+noun; wizard generate = **生成初稿**) |
| Empty state heading | 还没有项目 |
| Empty state body | 从一个技术想法开始，生成你的第一份专利初稿。 → 按钮「开始撰写」 |
| Error state | 操作失败：{原因}。请检查后重试，或在运行日志中查看详情。 |

**Destructive actions in this app (require `alert-dialog` confirm):**
1. 删除项目 (`deleteProject`)
2. 删除发明点 (`deleteProjectPatentPoint`)
3. 取消运行 (`cancel*Run`) — reversible; use a lightweight `dialog`, not full destructive styling.
