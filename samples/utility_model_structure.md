# V1 Smoke Sample: Utility Model Structure

## Patent path

从结构方案撰写实用新型（utility_model）。

## Structure idea

一种可调节边缘 AI 摄像机防护安装支架，包括固定底座、旋转支撑臂、俯仰调节座、防护罩、理线槽和快拆锁止件，用于在施工现场临边、塔吊基础和材料堆场等位置快速安装边缘摄像机。

## Components

1. 固定底座：带有膨胀螺栓孔和防松垫片，可固定在立柱、围挡或墙面上。
2. 旋转支撑臂：通过水平转轴连接固定底座，支撑臂上设置角度刻度和限位槽。
3. 俯仰调节座：通过弧形滑槽与旋转支撑臂连接，允许摄像机在预设角度范围内俯仰调节。
4. 防护罩：罩设于摄像机外侧，前端设置透明保护窗，顶部设置导水坡。
5. 理线槽：沿支撑臂布置，容纳电源线和网线，并通过卡扣盖板闭合。
6. 快拆锁止件：包括弹性插销和锁紧螺母，用于在维护时快速释放摄像机安装座。

## Structural effects

- 支架通过水平旋转和俯仰滑槽实现摄像机视角调节。
- 防护罩和导水坡降低雨水、灰尘对摄像机的影响。
- 理线槽减少施工现场线缆外露和被拉拽的风险。
- 快拆锁止件便于安全员在不拆除底座的情况下维护摄像机。

## Evidence state

- 可行未验证：快拆锁止件的疲劳寿命、透明保护窗在粉尘环境中的清洁周期。
- 需实验：不同安装基面上的抗振性能。

## Release-smoke expectations

该样例用于 v1 接受性测试中的实用新型轻量路径：无需严格多 Agent 会审和算法公式包即可生成结构化权利要求与说明书，然后进入正式稿编译、成稿会审和导出。测试不得调用真实 LLM API，也不得依赖任何 `.env` 或凭证。
